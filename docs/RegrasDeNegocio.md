# Regras de Negócio

Regras que governam o comportamento do jogo. Devem ser respeitadas no código e validadas durante o desenvolvimento.

---

| ID   | Regra                                                                                     |
| :--- | :---------------------------------------------------------------------------------------- |
| RN01 | Golem só executa tarefas se tiver uma runa equipada.                                      |
| RN02 | Runas só são obtidas completando dungeons.                                                |
| RN03 | Recursos coletados vão diretamente para o inventário do jogador.                          |
| RN04 | Golem pausa automaticamente se o inventário do jogador ou o Armazém estiverem cheios.     |
| RN05 | Golems do tipo Transportador movem recursos do mapa para o Armazém (ou baús — a definir). |
| RN06 | O Armazém tem capacidade limitada (valor a definir).                                      |
| RN07 | Cada tile do mapa pode conter apenas 1 construção.                                        |
